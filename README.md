# E-commerce Sales Dashboard

## Overview
This project is a data analysis dashboard built using Google Sheets. It analyzes e-commerce sales data to identify trends, performance, and key business insights.

## Dataset Source
The dataset used in this project is sourced from Kaggle:

E-commerce Sales Performance Analysis Dataset  
https://www.kaggle.com/datasets/uzmaakhtar/e-commerce-sales-performance-analysis

## Features
- Sales analysis by region
- Monthly sales trend visualization
- Top-performing products identification
- Category-wise sales distribution
- KPI metrics (Total Sales, Total Profit, Total Orders)

## Tools Used
- Google Sheets
- Pivot Tables
- Charts & Data Visualization

## Key Insights
- West region generates the highest sales
- Electronics category contributes the most revenue
- Sales show clear monthly trends over time
- Top-performing products drive a significant portion of revenue
